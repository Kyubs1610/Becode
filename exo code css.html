Digital Sprint - CSS Exercices

A chaque exercice, vous devrez réaliser quelques missions, après les modules cités du cours SoloLearn


1 - Après les modules "Basics" et "Working with text"

  1. Mettre la ligne indiquée dans le code CSS en commentaire.
  2. Une police est importée dans la balise "head" => mettre toute la page dans cette police.
  3. Mettre ce texte-ci en gras, et diminuer sa taille.
  4. Mettre ce texte en italique et en bleu (spécifier la couleur par son nom).
  5. Augmenter la taille de ce texte-ci, et le mettre en vert (spécifier la couleur par code hex).
  6. Centrer ce texte, et le mettre en rouge (spécifier la couleur en mode rgb).
  7. Souligner ce texte, augmenter l'espace entre ses mots et diminuer l'espace entre ses lettres.
  8. Mettre une ombre portée grise à ce texte-ci.
